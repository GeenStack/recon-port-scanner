# Changelog

## [1.0.0] - 2026-05-05

### Добавлено
- Быстрое сканирование портов с поддержкой masscan и nmap
- Детальное сканирование с определением версий сервисов
- Интеграция vulners.nse для поиска уязвимостей
- Сканирование с безопасными vuln скриптами nmap
- Генерация итогового отчета в формате JSON
- Поддержка входных данных от recon-subdomains-enumeration
- Поддержка простого списка IP адресов
- Сохранение всех логов сканирования для каждого хоста
- Docker контейнеризация
- Настройка через .env файл
- Опция принудительного сканирования без ping
- Настраиваемый диапазон портов
- Настраиваемая скорость сканирования masscan
- Полная документация на русском языке

### Особенности
- Многоэтапное сканирование с автоматической оркестрацией
- Детальные логи для каждого этапа сканирования
- Структурированный JSON отчет с информацией о портах, сервисах и уязвимостях
- Ссылки на файлы логов в итоговом отчете
